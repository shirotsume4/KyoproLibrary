# 座圧

与えられた1次元配列を座標圧縮します。
つまり、与えられた非負整数列 A = [A_1, A_2, ... A_N] に対して、以下の条件を満たす数列 B を返します。

B の要素は全て非負整数。

任意の (i, j) (0 <= i < j < n) に対して、 A_i > A_j なら B_i > B_j 、A_i = A_j なら B_i = B_j 、 A_i < A_j なら B_i < B_j

上の条件を満たす中で、 max(B) は最小。
